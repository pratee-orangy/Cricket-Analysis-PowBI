# Cricket-Analysis-PowBI
Power BI dashboard analyzing ODI batting performance using ESPN Cricinfo data, focusing on player consistency and impact in away matches.

1.Project Overview
This project analyzes Indian batsmen's ODI performance in away matches against Australia using ESPN Cricinfo data.

2.Business Objective
To provide data-driven insights into player consistency, impact and performance in challenging away conditions.

3.KPIs
- Total Runs
- Batting Average
- Strike Rate
- Runs per Match
- Highest Score
- Consistency Index
  
4.Tools & Technologies
- Power BI Desktop
- Power Query (M Language)
- DAX
- Excel , CSV
  
5.Data Processing
- Data extracted from ESPN Cricinfo
- Cleaned and transformed using Power Query
- Modeled using star schema concepts
  
6.Dashboard 
- Interactive player comparison
- Performance KPIs
- Consistency analysis
- Dynamic slicers
  
7.Repository Structure
- Data: Raw dataset
- PowerBI: PBIX report
- Screenshots: Dashboard previews
  
8. Outcome
Provides actionable insights for cricket analysts and selectors to evaluate player performance objectively.
